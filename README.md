# Flask CRUD Application

This is a basic CRUD (Create, Read, Update, Delete) project built using Flask and Python.

## Features

- Create Data
- Read Data
- Update Data
- Delete Data

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite

## CRUD Operations

### Create
Add new data into the database.

### Read
Display stored data.

### Update
Edit existing data.

### Delete
Remove data from the database.

## How to Run Project

1. Clone this repository

```bash
git clone <your-github-repo-link>
```

2. Open project folder

```bash
cd project-name
```

3. Install Flask

```bash
pip install flask
```

4. Run the project

```bash
python app.py
```

5. Open browser and visit

```bash
http://127.0.0.1:5000/
```
